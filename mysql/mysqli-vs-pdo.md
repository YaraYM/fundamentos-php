# mysqli x PDO

- O mysqli é uma extensão do próprio PHP para conectar ao banco MySQL e tem uma proximidade do código nativo, sendo mais rápida do que o PDO
- O PDO é uma API para conexão de banco de dados, não limitada ao MySQL, que abstrai alguns conceitos com código PHP, tornando esta abordagem mais lenta
- Ambas as formas podem ser utilizadas de forma orientada a objetos
