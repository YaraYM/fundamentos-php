# Teoria do processamento de formulários

- Vamos realizar processamentos de formulário de duas maneiras: via GET  e via POST
- Com o GET, vamos processar os parâmetros que vêm na query string, ou seja, na URL
- Com o POST, vamos processar as informações que vêm na requisição; estas não aparecem na URL
- Exemplos de uso: Buscar => GET, Registros de usuário => POST
