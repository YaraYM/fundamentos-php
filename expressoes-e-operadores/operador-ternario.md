# Operador Ternário

- Este operador constitui uma estrutura de condição resumida.
- Na maioria dos casos, vamos optar por if/else.
- Porém, em situações simples, podemos utilizar o ternário.
- Exemplo: 5 > 2 ? echo "5 é maior que 2" : echo "5 é menor que 2";
- A primeira interrogação vem antes da comparação e o : é utilizado para uma segunda situação, caso a primeira seja falsa.
