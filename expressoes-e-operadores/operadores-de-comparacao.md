# Operadores de Comparação

- As operações com operadores de comparação resultarão em true or false.
- Igualdade: ==
- Idêntico a: ===
- Diferença: !=
- Não idêntico a: !==
- Maior e maior ou igual a: > e >=
- Menor e menor ou igual a: < e <=

## Operador de igualdade

- Com o operador de igualdade, verificamos se um valor é igual ao outro.
- O símbolo é: ==
- Exemplo: 5 == 4 # false
- Exemplo: 3 == 3 # true

## Operador idêntico a 

- Com o operador idêntico a, verificamos se um valor é igual ao outro, levando também em consideração seu tipo.
- O símbolo é: ===
- Exemplo: 5 === 5 # true
- Exemplo: 3 === "3" # false

## Operador de diferença

- Com o operador de diferença, verificamos se um valor é diferente de outro.
- O símbolo é: !=
- Exemplo: 5 != 5 # false
- Exemplo: 10 != 5 # true

## Operador não idêntico a 

- Com o operador 'não idêntico a', verificamos se um valor é diferente de outro, avaliando também seu  tipo.
- O símbolo é: !==
- Exemplo: 5 !== 4 # false
- Exemplo: 3 !== "3" # true

## Operador maior e maior ou igual

- Com o operador 'maior que', verificamos se um valor é maior que outro.
- O símbolo é: >
- Exemplo: 5 > 4 # true
- Com o operador 'maior ou igual a', verificamos se um valor é maior ou igual a outro.
- O símbolo é: >=
- Exemplo: 5 >= 5 # true
