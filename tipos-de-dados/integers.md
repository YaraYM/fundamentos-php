# Inteiros (integers)

- Os inteiros são os números inteiros da matemática, como 1, 2, 15.
- Inclui os números negativos.
- Os números positivos não precisam de um sinal de + na frente; já os números negativos devem ser descritos assim: -1, -2, -15.

## Checando número inteiro

- Podemos validar se um dado é inteiro com a função is_int().
- Caso um número seja inteiro, será retornado true.
- Caso não seja, receberemos um retorno de false.
- Precisamos utilizar uma estrutura if para validar o valor.
